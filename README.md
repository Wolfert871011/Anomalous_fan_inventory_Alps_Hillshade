# Anomalous_fan_inventory_Alps_Hillshade
Supporting webmap for "Geomorphology Article Alpine megafans, outsize fans, and lesser anomalous fans  - a first comprehensive inventory for the European Alps" by Thomas Wolfert and David Jarman

Leaflet map for opening in browser

1) Content:

a) Anomalous fan inventory on ESRI Satellite basemap
b) Anomalous fan inventory on ESRI Hillshade basemap
c) Excel version of inventory
d) Original kmz

2) Installation:

a) Extract the compressed dataset into preferred folder
b) Open the index.html file of one of the datasets with browser (Microsoft Edge, Google Chrome, Mozilla Firefox,...)

3) Usage:

a) Clicking on individual point opens window with associated attributes (key below)
b) Zoom in or out with mouse or +/- buttons in upper left corner
c) Measure distances or areas with measuring tool in upper left corner
d) Search fan codes (e. g. AV1) with looking glass icon in upper left corner

4) Key for attributes

Code => Code of fan
Fan => Name of fan
M/OF => Megafan or outsize fan
LMS => Lost mountain source -> 0 = no obvious source 1 = reconstruction sketched 2 = reconstruction easily visualized 3 = no obvious reconstructable cavity but options evident
RSF => Rock slope failure occurrence -> 0 = no indications in vicinity 1 = RSD strong evidence 2 = RSD weaker evidence 3 = RS substantial rockslides 4 = RAv rock avalanches
cavity =>  Cavity/Catchment form -> 1 = leaf 2 = rectilinear 3 = wedge/slot 4 = complex 5 = multiple
debouchure => 1 = hourglass 2 = chute 3 = incised 4 = gorge 5 = unclassifiable
morpho-loc => morpholocation -> 1 = flank 2 = truncated spur 3 = hanging 4 = small tributary 5 = large tributary 6 = upper valley 7 = island hill / peak
breach => 1 = within a breach 2 = within 10 - 15 km of a breach locus
cata/allom => allometric-catastrophic -> 1 = allometric, 2 = hybrid, 3 = catastrophic
fan type => fan form -> 1 = classic 2 = wide 3 = multiple 4 = mis-shape 5 = talus nb = alluvial/hybrid
fan extens => fan extension -> 0 = nil 1 = run on 2 = back-up 3 = both
Fan height => in m
Fan radius => in m
Fan gradie => Fan gradient in %
Fan area => in km²
Catchment => Catchment area in km²
Fan area_ => Fan area/ Catchment area ratio
Cone secto => Cone sector average by eye in 5% steps
Proxy Volu => Proxy volume
Tier => Tier per thresholds
Volume_Ca => Volume/Catchment area volume
Latitude Y => Latitude in m (EPSG 3035)
Longitude => Longitude in m (EPSG 3035)
Elevation => Terrain elevation in m a.s.l
Aspect => in degrees
Geology => Basic geological class
Dam =>    O –  DU distally unconfined ie fan toe does not reach far side of valley, fades into floor, does not (greatly) displace river;   1 -  DC  distally confined – toe just reaches far side, displaces river to opposite slope foot;   2-   toe fluvially incised – minor ravine;  3 - toe fluvially incised – major ravine;   4 – fan retains lake / wetland / flat infill (lake floor/alluvial fill)’
